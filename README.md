# sound_manager

A Flutter plugin for recording and playing audio files.

## What's in the box
- Record audio
- Pause and resume recording (Android API level 24>)
- Save recordings in 3gp format
- Play audio files
- Pause and resume playing audio files
- Seek to position
- Audio looping
